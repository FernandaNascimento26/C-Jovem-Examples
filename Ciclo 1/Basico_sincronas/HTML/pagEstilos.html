CSS inline

Texto com estilo inline. Descrição: O CSS inline é aplicado diretamente no elemento HTML usando o atributo style.
Quando usar: Ideal para pequenos ajustes específicos que não se repetem, mas não recomendado para grandes projetos, pois dificulta a manutenção.

CSS Interno (embutido)

Descrição: O CSS interno é colocado dentro da própria página HTML, em uma tag style dentro do head.
Quando usar: Útil para páginas simples ou quando os estilos são exclusivos de uma única página.
