1. Layout e Estrutura

Propriedades para definir a estrutura dos elementos na página.

display: Define o tipo de exibição do elemento (ex.: block, inline, flex).

position: Define o método de posicionamento do elemento (ex.: absolute, relative, fixed).

float: Faz o elemento flutuar à esquerda ou à direita (ex.: left, right).

2. Espaçamento

Propriedades para controlar o espaço ao redor e dentro dos elementos.

margin: Define o espaço externo ao redor do elemento (ex.: 20px, auto).

padding: Define o espaço interno entre o conteúdo e a borda do elemento (ex.: 15px).

border: Define a borda do elemento (ex.: 1px solid black).

3. Tipografia e Formatação de Texto

Propriedades para definir a aparência do texto.

font-size: Define o tamanho da fonte (ex.: 16px).

font-family: Define o tipo de fonte (ex.: Arial, serif).

text-align: Alinha o texto horizontalmente (ex.: center).

4. Cores e Plano de Fundo

Propriedades para definir a cor do texto e o fundo dos elementos.

color: Define a cor do texto (ex.: blue, #ff0000).

background-color: Define a cor de fundo (ex.: #f0f0f0).

background-image: Define uma imagem de fundo (ex.: url('imagem.jpg')).

5. Bordas e Formas

Propriedades para definir bordas e o arredondamento dos cantos.

border: Define a borda ao redor do elemento (ex.: 1px solid black).

border-radius: Define o arredondamento dos cantos (ex.: 10px).

6. Tamanho e Dimensões

Propriedades para ajustar a largura e altura dos elementos.

width: Define a largura do elemento (ex.: 100%, 500px).

height: Define a altura do elemento (ex.: 50vh, 300px).

7. Efeitos Visuais e Transformações

Propriedades para aplicar sombras, opacidade e transformações.

opacity: Define a transparência do elemento (ex.: 0.5).

box-shadow: Adiciona sombra ao redor do elemento (ex.: 2px 4px 8px rgba(0,0,0,0.5)).

transform: Aplica transformações como rotação ou escala (ex.: rotate(45deg)).

8. Posicionamento e Exibição Responsiva

Propriedades para adaptar a exibição em diferentes tamanhos de tela.

media queries: Adapta estilos para diferentes resoluções (ex.: @media (max-width: 600px)).

viewport units: Unidades relativas à tela (ex.: vw, vh).
